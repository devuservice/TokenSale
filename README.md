## ICO UST Tokens Contracts ##

This repository contains Solidity smart contracts.

### ManagedToken.sol ###

This smart contract contains an implementation of the following functions:
* ERC20 Token Standard
* mint/demint token
* change Name and Ticker
* lock/unlock transfer of tokens
* set new contract for crowdsale of tokens
* set manager for manual transfer of token
* upgrade of contract
